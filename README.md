# SwiftVerify

A gift card verification web app with role-based authentication, real-time card validation, and an admin management panel.

## Features

- Role-based access control — multiple user types with different permissions
- Gift card upload and verification workflow
- Cloudinary integration for secure image storage
- Superadmin settings panel for platform management
- Custom domain deployment

## Built With

- **Frontend** — HTML, CSS, JavaScript
- **Backend** — Node.js, Express
- **Database** — MySQL
- **Storage** — Cloudinary
- **Deployment** — Render
